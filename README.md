# 🚀 Agency AI — Modern React Landing Page

A modern **AI Agency Landing Page** built using **React + Vite + Tailwind CSS** with smooth animations and clean UI design.

🌐 **Live Demo:**
https://sandeepkumarmamloda.github.io/agency-ai/

---

## ✨ Features

* ⚡ Fast performance powered by Vite
* 🎨 Modern UI design
* 📱 Fully responsive layout
* 🌙 Clean and minimal interface
* 🎬 Smooth animations using Motion
* 💡 Component-based architecture

---

## 🛠️ Tech Stack

* **React 19**
* **Vite**
* **Tailwind CSS**
* **Motion Animation**
* **JavaScript (ES6+)**

## 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/sandeepkumarmamloda/agency-ai.git
```

Go into the project folder:

```bash
cd agency-ai
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview build:

```bash
npm run preview
```

---

## 🌍 Deployment

The project is deployed using **GitHub Pages**.

```bash
npm run deploy
```
